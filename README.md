# InventoryViewer

## Description
プレイヤーのインベントリをGUIを使用して閲覧することができます。

## Usage
- コマンド:/inventoryviewer
- 短縮:/inview

## Notes
このSkriptファイルを使用するには、**SkQuery**が必要です！

## Version
- minecraft:1.12.2
